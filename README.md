# India Census Forecast & Projections Dashboard
An interactive, full-stack demographic analysis and visualization dashboard built with **Django**, **PostgreSQL**, and **Chart.js**. The platform enables granular analysis of legacy Indian Census datasets (2001 and 2011) and generates dynamic growth forecasting models up to the year 2026 at the national, state, and district levels.
*Developed as part of an internship project at the **Research Designs and Standards Organisation (RDSO), Indian Railways**.*
---
## 🚀 Key Features
* **Multi-Scope Geography Filtering**: Explore census metrics seamlessly across three levels: Total (All India), State, and District.
* **Demographic Projections (2012–2026)**: Computes real-time Compound Annual Growth Rates (CAGR) and models future population trends over a 15-year forecast window.
* **Interactive Visualizations**:
  * **Line Graphs**: Time-series population forecasts (2001–2026).
  * **Pie Charts**: Gender distribution comparison between 2001 and 2011.
  * **Bar Charts**: Age distribution breakdowns comparison across census years.
* **Modern glassmorphism UI**: Beautiful dark theme built with modern CSS variables, layout transitions, and dynamic dropdown menu selectors.
* **Automated Data Pipeline**: Seamless pipeline to parse, clean, and import raw census CSV data into a PostgreSQL relational database.
---
## 🛠️ Technology Stack
* **Backend**: Django (Python Web Framework)
* **Database**: PostgreSQL (Relational Database)
* **Data Processing & ETL**: Pandas, SQLAlchemy
* **Frontend**: HTML5, Vanilla CSS3 (Custom Glassmorphic Theme), Vanilla JavaScript
* **Charts**: Chart.js
* **Testing**: Django Test Client
